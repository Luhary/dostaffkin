# DoStaffKin

Учебный проект, разработанный в рамках интенсива по Angular.

## О проекте

DoStaffKin — веб-приложение, созданное для практики разработки современных SPA-приложений с использованием Angular и TypeScript.

Проект демонстрирует навыки работы с компонентным подходом, маршрутизацией, управлением состоянием интерфейса и адаптивной версткой.

## Демо

🔗 Live Demo: https://luhary.github.io/dostaffkin/

* Современный адаптивный интерфейс
* Компонентная архитектура
* Маршрутизация между страницами
* Работа с данными
* Оптимизированная структура проекта

## Установка и запуск

```bash
git clone https://github.com/username/dostaffkin.git

cd dostaffkin

npm install

ng serve
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:4200
```

## Сборка проекта

```bash
ng build
```
